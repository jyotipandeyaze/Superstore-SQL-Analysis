# 📊 Superstore SQL Analysis

## 📌 Project Overview
This project analyzes retail sales, profitability, customer trends, and regional performance using SQL on the Superstore dataset.

The objective was to generate business insights and perform KPI-based analysis for decision-making.

---

## 🛠️ Tools Used
- SQL
- DB Browser for SQLite
- Kaggle Superstore Dataset
- GitHub

---

## 📈 Business Questions Solved

1. Which region generates the highest sales?
2. Which category is the most profitable?
3. Which products generate losses?
4. Who are the top customers by revenue?
5. What are the monthly sales trends?
6. Which shipping mode is most used?

---

- GROUP BY

- ORDER BY

- INNER JOIN
